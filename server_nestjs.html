CORS on NestJS

In your NestJS app on node.js, do the following with your application: 

app.enableCors({
  origin: "YOUR-DOMAIN.TLD", // update to match the domain you will make the request from
  allowedHeaders: "Origin, X-Requested-With, Content-Type, Accept"
});

Alternatively, use the options object of the factory's create() method:

const app = await NestFactory.create(AppModule, { cors: true }); // to use defaults

// or, for full configuration control:
const app = await NestFactory.create(AppModule, {
  cors: cors: {
    origin: "YOUR-DOMAIN.TLD", // update to match the domain you will make the request from
    allowedHeaders: "Origin, X-Requested-With, Content-Type, Accept"
  }
});
